SRI Hash Generator

This is a Node.js CLI tool that must be run from the terminal.

Command:
node tools/generate-sri-hashes.js

Requirements:

  • • Node.js installed
  • • Internet connection (fetches from CDNs)

After running, copy the generated integrity attributes to the <script> tags in index.html
