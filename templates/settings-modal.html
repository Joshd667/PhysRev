The CSV to JSON Converter creates an optimized data file that dramatically improves app loading speed:

  • Reduces loading time to sub-second speeds
  • Combines all data into a single HTTP request
  • Pre-processes data for instant access
  • Includes groups configuration (no CSV fallback)

Place the generated combined-data.json file in the resources/ directory for optimal performance.
